# 提案 Intent

## 這是什麼

一款可在瀏覽器直接遊玩的 **經典回合制 RPG**，採用 **2D 卡通 / 立繪** 風格。玩家透過網頁打開遊戲，無需下載安裝，即可體驗類似《勇者鬥惡龍》、《Final Fantasy》早期作品的回合制冒險樂趣：地圖探索、隨機遇敵、回合制戰鬥、等級成長、裝備與道具系統。

## 為誰而做

- **核心目標**：自己（作者）作為開發練習與創作作品。
- **潛在玩家**：喜歡日式古典 RPG、懷舊玩家、不想下載安裝大型遊戲的休閒玩家、想在工作空檔玩個 10–20 分鐘的人。

## 解決什麼問題

現在多數 RPG 需要下載 GB 級別客戶端或註冊帳號。本作以「**打開瀏覽器即玩**」為訴求，提供低門檻、可分享 URL 給朋友體驗的回合制冒險。玩家不必擔心裝置相容性，只要有現代瀏覽器（Chrome / Firefox / Safari / Edge）即可遊玩。同時這也是作者實驗 web 遊戲開發技術棧（Canvas / WebGL / 狀態管理 / 存檔機制）的載體。

## 初步輪廓（後續階段細化）

- **核心玩法**：地圖探索 → 隨機遇敵 → 回合制戰鬥（攻擊 / 魔法 / 道具 / 防禦 / 逃跑）→ 經驗值與等級成長 → 商店買賣 → 主線劇情推進
- **視覺呈現**：2D 卡通角色立繪（戰鬥畫面） + 俯視角地圖（探索畫面）
- **平台**：純前端網頁應用，無需後端（存檔用 LocalStorage / IndexedDB）
- **單人遊戲**，無多人連線需求
