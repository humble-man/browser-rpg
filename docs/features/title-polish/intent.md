# 提案 Intent — Title screen polish

## 這是什麼

把標題畫面從「功能列表」升級為「有氛圍的入口」：

```
┌────────────────────────────┐
│                            │
│      ⚔️ 幽影迷宮 ⚔️         │
│   ─ 碧楓村的傳說 ─         │
│                            │
│    [🗡️ 開始新遊戲]         │
│    [📂 載入存檔]           │
│    [📥 匯入存檔]           │
│    [🗑️ 刪除存檔]           │
│                            │
│    [🔊 聲音]               │
│                            │
│    v0.12.0 · GitHub        │
└────────────────────────────┘
```

新增：
- Subtitle「碧楓村的傳說」
- 標題上下的裝飾線
- 音效切換鈕（沿用 cycle 14 setMuted/isMuted API）
- 底部版本號 + GitHub repo 連結

## 為誰而做

- **新玩家**：第一印象有氛圍，不只是 4 個按鈕
- **想 mute 但未進遊戲的玩家**：標題即可切換音效
- **找版本資訊的玩家**：腳註顯示版本 + repo link

## 解決什麼問題

當前標題畫面 cycle 1 設計，4 個功能按鈕一字排開，缺：
- 遊戲氛圍 / 視覺層次
- 標題畫面就能設定音效（cycle 14 mute 入口只在 ESC 選單）
- 版本資訊（玩家不知道自己玩到哪個版本）

## 範圈

- ✅ Subtitle「碧楓村的傳說」
- ✅ 裝飾線 + 視覺層次調整
- ✅ 音效切換鈕（與 ESC 選單同樣的 toggle）
- ✅ 底部 footer：v0.12.0 + GitHub link
- ✅ 版本號從 `import.meta.env.PACKAGE_VERSION` 或 vite define
- ❌ 不改 4 個既有功能按鈕（新遊戲 / 載入 / 匯入 / 刪除）
- ❌ 不加 settings panel（過度範圍，cycle 19+）
- ❌ 不加 BGM 觸發（cycle 14 SE 已支援、BGM 留未來）
- ❌ 不改其他 scenes
